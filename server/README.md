# A股爬虫监控

## 数据源
- 新浪
- 同花顺
- 网易

## 使用库
- springboot
- mybatis
- httpclient

## REST API接口
- 查询N天内连续涨停跌停数据:<br/>
http://127.0.0.1:5000/rest/stock/limitUpDownList/2021-03-16/10

- 查询N天内连续涨跌数据:<br/>
http://127.0.0.1:5000/rest/stock/upDownList/2021-03-16/10/1

- 查询N天内从从下跌到上涨趋势的数据:<br/>
http://127.0.0.1:5000/rest/stock/upward/2021-03-16/10

- 放量数据列表:<br/>
http://127.0.0.1:5000/rest/stock/recommandList/2021-03-16/1

- 添加黑名单列表:<br/>
http://127.0.0.1:5000/rest/stock/addBackList?codeList=sh600000,sz000001

- 抓取股票数据列表:<br/>
http://127.0.0.1:5000/rest/stock/crewQuotationList/1

- 执行单次任务:<br/>
http://127.0.0.1:5000/rest/stock/singleTask

- 执行最终任务:<br/>
http://127.0.0.1:5000/rest/stock/finalTask

## 新浪返回值含义
- 0：”大秦铁路”，股票名字；
- 1：”27.55″，今日开盘价；
- 2：”27.25″，昨日收盘价；
- 3：”26.91″，当前价格；
- 4：”27.55″，今日最高价；
- 5：”26.20″，今日最低价；
- 6：”26.91″，竞买价，即“买一”报价；
- 7：”26.92″，竞卖价，即“卖一”报价；
- 8：”22114263″，成交的股票数，由于股票交易以一百股为基本单位，所以在使用时，通常把该值除以一百；
- 9：”589824680″，成交金额，单位为“元”，为了一目了然，通常以“万元”为成交金额的单位，所以通常把该值除以一万；
- 10：”4695″，“买一”申请4695股，即47手；
- 11：”26.91″，“买一”报价；
- 12：”57590″，“买二”
- 13：”26.90″，“买二”
- 14：”14700″，“买三”
- 15：”26.89″，“买三”
- 16：”14300″，“买四”
- 17：”26.88″，“买四”
- 18：”15100″，“买五”
- 19：”26.87″，“买五”
- 20：”3100″，“卖一”申报3100股，即31手；
- 21：”26.92″，“卖一”报价
- (22, 23), (24, 25), (26,27), (28, 29)分别为“卖二”至“卖四的情况”
- 30：”2008-01-11″，日期；
- 31：”15:05:32″，时间；
